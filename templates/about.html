Indo Sentiment

Sebuah website pencarian tweet atau post social media dari Twitter yang menampilkan tweet terbaru atau berdasarkan Daterange dan visualisasi Pie Chart dengan sentimentnya berupa positif, negatif dan netral. Website ini dibuat menggunakan bahasa pemrograman Python dengan perhitungan probabilitas sentiment menggunakan algoritma Naive Bayes dari library Textblob, engine backend menggunakan library Flask, Data API menggunakan Tweepy dan option stemming menggunakan library Sastrawi. Dataset ini menggunakan tweet mengenai pilgub jawa barat 2018, sehingga website pencarian ini lebih mengutamakan sentiment mengenai politik.
